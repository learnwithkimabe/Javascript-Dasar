• Saat bikin aplikasi, kadang kita input dari pengguna selalu alam bentuk string
  • Sedangkan misal kita ingin mengelola datanya dalam bentuk number
  • Jadi perlu untuk dilakukan konversi data
  • parseInt(string) : Mengkonversi dari string ke number (bilangan bulat)
  • parseFloat(string) : Mengkonversi dari string ke number (bilangan pecahan)
  • number(string) : Mengkonversi dari string ke number (bilangan bulat / pecahan)
  • number.toString() : Mengkonversi dari number ke string
  • Dalam mengkonversi ketika data tidak sesuai seperti misal text berupa kata ke number maka hasilnya akan NaN
  • untuk metode parseInt dan parseFloat ketika data yang akan di konversi masih memiliki angka, maka angka tersebut akan ditampilkan dengan syarat angka berada di depan huruf
  • untuk Number meskipun data text memiliki angka didalamnya, hasilnya akan tetap NaN apalagi string tanpa angka sama sekali
  • isNaN adalah funtion untuk mengecek apakah data sebuah number itu NaN atau bukan
  • isNaN menghasilkan nilai boolean, true atau false
