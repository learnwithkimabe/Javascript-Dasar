• Di JS hanya mendukung 1 tipe data number. Yaitu bisa bilangan bulat (pecahan) atau desimal
  • Tipe data itu bisa diartikan jenis data yang akan kita input sesuai jenis jenisnya.
  • Kalo ingin menginput tipe data number tinggal masukan angka tanpa tanda kutip "100" karena setiap tanda kutip akan dibaca sebagai tipe data string (jenis text)

basis angka defaultnya 10, JS sudah mendukung binary hexadecimal dan octal

  • Hexadecimal : 0xFF
  • Binary : 0b****
  • Octal : 0o10
