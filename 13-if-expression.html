• Di JS, if adalah kata kunci untuk percabangan
  • percabangan artinya mengeksekusi kode tertentu ketika suatu kondisi terpenuhi
  • kondisi menghasilkan boolean
  • kata kuncinya adalah if
  • Blok if akan dieksekusi ketika kondisi bernilai TRUE
  • Kadang kita ingin melakukan eksekusi program tertentu jika kondisi FALSE
  • hal ini bisa dilakukan menggunakan else expression
  • Kadang dalam if kita butuh beberapa kondisi
  • Kita bisa menggunakan elfe if expression untuk menciptakan kondisi alternatif
