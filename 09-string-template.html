• String template bisa mempermudah menggabungkan sebuah string yg dibuat dalam variable dengan memanggil nama variable tanpa harus menggunakan operator aritmatika lagi seperti cara sebelumnya.
  • Untuk menggunakan string template yaitu dengan menggunakan backtick (`) bukan tanda kutip ' atau "
  • Lebih fleksibel untuk enter, tab langsung (multiline string) dan bisa mengurangi penggunaan escape sequence. Namun penulisan pada script code sangat berpengaruh.
  • untuk memanggil nama variable maka tambahkan ${nama_variable}
  • Bisa di kombinasikan dengan expression string (perbandingan)
