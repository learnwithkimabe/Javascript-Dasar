• tanda + : untuk operasi penambahan
  • tanda - : untuk operasi pengurangan
  • tanda * : untuk operasi perkalian
  • tanda ** : untuk operasi perpangkatan (exponensial)
  • tanda / : untuk operasi pembagian
  • tanda % : untuk operasi modulus (sisa bagi)

untuk melakukan operasi matematika dengan varible dirinya sendiri

misalnya : let angka = 10

angka + 2 = 12

bisa dilakukan dengan cara angka += 2 maka hasilnya sama.

  • Operator unary + : untuk menandakan nilai positif (default)
  • Operator unary - : untuk menandakan nilai negatif
  • Operator unary ++ : untuk menaikan nilai 1 angka
  • Operator unary -- : untuk menurunkan nilai 1 angka
