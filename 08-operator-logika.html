Operator logika a/ operator untuk 2 buah data boolean

Hasil dari operator logika adalah boolean lagi.

  • && : Dan (akan mengasilkan nilai true jika 2 nilai yg dibandingkan bernilai true) kalau salah satunya atau keduanya false hasilnya false
  • || : Atau (akan menghasilkan nilai true, meskipun nilai yang dibandingkan salah satunya atau keduanya bernilai true), hanya akan false jika kedua data yg dibandingkan bernilai false.
  • ! : Kebalikan (Hanya digunakan untuk 1 data untuk mengubah nilai yg tadinya true jadi false atau sebaliknya)
