• Array adalah type data yang berisikan kumpulan data. Bebas bisa angka, text, boolean, object, function dll
  • Array sifatnya dinamis, artinya data bisa bertambah dengan sendirinya saat kita memasukkan data ke dalam array
  • Array memiliki nomor index, dengan diawali index 0
  • jika ingin membuat array gunakan [ ] pada variablenya
  • jka ingin memasukkan data lebih dari 1, pisahkan dengan koma (,)
  • Seperti yg sudah disampaikan sebelumnya bahwa array dimulai dari index 0
  • dan jika menambahkan data ke dalam array, otomatis data akan disimpanat urutan terakhir
  • urutan data pada array di sebut index
  • Bahkan di dalam array kita bisa menambahkan array juga.
  • array.push(value) : untuk menambah data array
  • array.length : untuk melihat jumlah data array
  • array[no.index] : untuk melihat data array pada nomor index ke berapa
  • array[no.index] = value : untuk mengubah data dalam array dengan posisi nomor index tertentu
  • delete array[no.index] = value : untuk menghapus data dalam array dengan posisi nomor index tertentu, namun index tidak megnurangi array.length nya. Artinya length array masih seperti awal, hanya saja index yang dihapus valuenya undefined
