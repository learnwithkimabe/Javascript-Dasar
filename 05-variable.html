• Variable a/ tempat untuk menyimpan data
  • Dengan kita menyimpan data di variable, kita bisa menggunakan data dengan hanya memanggil nama variablenya
  • Membuat var bisa dengan kata kunci var, let, const
  • JS a/ dynamic languange, memungkinkan kita untuk mengubah type data (tidak terpaku dengan 1 type data) pada 1 variable
  • Variable bisa digunakan berkali kali diakses
  • Mempermudah untuk memanggil data sama berulang ulang
  • Cukup dengan memanggil nama variblenya saja

Kata kunci lain untuk membuat variable. Ini lebih di rekomendasikan untuk versi terbaru javascript. (tidak disarankan untuk menggunakan var lagi.

  • let : digunakan untuk variable yang valuenya bisa kita ubah setelah di deklarasikan.
  • const : digunakan untuk variable yang valuenya constant (tidak bisa diubah) setelah di deklarasikan.
