• Tipe data string a/ type data yang berisi text atau karakter kosong didalam tanda petik 2 (") atau 1 (')
  • di JS penulisan ini menggunakan " (petik dua) atau ' (petik satu) sebelum dan sesudah isi text
  • Jika penggunaan awal dan akhir berbeda, misal awal " dan akhir ' , ini akan menghasilkan eror. Akan membuat kode keseluruhan terdampak tidak bisa berjalan
  • String bisa digabungkan dengan menggunakan tanda + (plus) u/ menghubungkan 2 buat string (text)
  • Jadi jika digunakan pada string tanda operator aritmatika bukan digunakah untuk menghitung penjumlahan
  • Jika digunakan untuk string namanya concatenated operator

Escape sequence karakter khusus seperti Enter, Tab, atau tanda quotes yang ingin ditampilkan.

  • Jika ingin enter pada kalimat misalnya, maka : \n
  • Jika ingin tab pada kalimat misalnya, maka : \t
  • Jika ingin tanda ' pada kalimat misalnya, maka : \'
  • Jika ingin tanda " pada kalimat misalnya, maka : \"
  • Jika ingin tanda / pada kalimat misalnya, maka : \/
